🛸 The Paradigm Shift: From Cascading to Unified Live Voice

Legacy Cascaded Pipeline

❌ Traditional STT/Translation Cascade

Stacking multiple independent APIs together introduces high cumulative network lag, high tokens cost, and loses all vocal tone and conversational context:

🎙️ Vocal Input → User speaks into microphone
📜 STT (Neural Captions) → Audio transcoded into text blocks
🤖 LLM Translation (Neural Flash) → Text is translated word-by-word
🔊 TTS Synthesis Engine → Text translated back into robotic voice
⏱️ Cumulative Latency: 4000ms+ (Too slow for emergency rooms)
Zero Gravity Agent Native

✅ Streamlined Voice-to-Voice

Completely eliminates the intermediates. Native bidirectional streaming delivers vocal nuances and instantaneous safety-checked interpretation:

🎙️ Vocal Stream → Low-latency PCM audio streamed via WebSockets
🛰️ Native Interpretation → Gemini 3.1 Live processes stream end-to-end
🚦 Semaphore Interceptor → ADK reasoning + Vertex RAG mid-utterance
🎧 Direct Vocal Out → Flawless natural Spanish/English vocal response
⏱️ Conversation Latency: <200ms (Real-time clinical co-pilot)
